# Monitoring

## Dashboard Design

The CloudWatch dashboard visualizes key system metrics.

Golden Signals monitored:

Traffic → RequestCount  
Errors → ErrorCount  
Saturation → CPUUtilization

## Widgets

RequestCount  
ErrorCount  
OrdersCreated  
OrderValue  
CPUUtilization

## Troubleshooting

When investigating issues:

1. Check dashboard metrics
2. Identify spikes or anomalies
3. Inspect logs for errors
